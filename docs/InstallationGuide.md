# AAC Installation Guide

## Programmable Block

Copy the complete contents of `src/AAC.cs` into one Space Engineers Programmable Block. The script is self-contained and must be compiled as a single file in the in-game editor.

## Tags

- `[AAC]` marks AAC-owned propulsion hardware.
- `[AAC] Flight` receives the flight page.
- `[AAC] Maintenance` receives maintenance and event information.
- `[AAC] Engineering` receives engineering or active debug pages.

## Debug Commands

Use `debug generators` for generator inspection and `debug mass` for artificial-mass inspection. Use `debug next` and `debug prev` to move within the active inspector.
