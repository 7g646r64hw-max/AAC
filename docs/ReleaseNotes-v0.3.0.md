# AAC Release Notes v0.3.0

Milestone 3 introduces automatic PEM-based propulsion assessment while preserving monitor-only safety.

## Highlights

- Automatic block orientation and gravity projection-axis detection.
- PEM-cached validation, contributing, redundancy, tolerance, and health data.
- READY assessment per ship axis.
- PEM summary pages with detected, tagged, contributing, and non-contributing counts.
- Generator and artificial-mass debug inspectors with stable component IDs.

## Compatibility

`AAC.cs` is intended to compile as a single Space Engineers Programmable Block script. No support files, partial classes, or external C# sources are required.
